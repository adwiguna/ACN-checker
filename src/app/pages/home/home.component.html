Welcome to Anggun Dwiguna Angular Application: Australian Company Number Checker

Create an Angular App under new repo in Incubator project to validate Australian Company Number (ACN) The ACN is a nine digit number with the last digit being a check digit calculated using a modified modulus 10 calculation. ASIC has adopted a convention of always printing and displaying the ACN in the format nnn nnn nnn, that is three blocks of three characters separated by a blank. This is to assist readability and the inserted blanks do not form part of the ACN.

Examples of valid ACNs include:

{{ validACN[index + key] }}

Acceptance Criteria

  • User able to input ACN using nnn nnn nnn format or nnnnnnnnn (without blank space)
  • The app able to validate the ACN correctly
  • Unit tests are present
